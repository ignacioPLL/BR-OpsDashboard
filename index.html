Update Monthly Data

Edit the values below and click Apply — all charts refresh automatically.

current_month: label shown in the header
kpi: current month — raw totals (invoices + order counts). The dashboard calculates per-order costs automatically.
prev_month: same fields for the previous month (for the delta arrows)
history: push a new entry each month to grow the trend charts
zones / carriers / transit_distribution: current month breakdown
